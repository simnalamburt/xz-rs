xz-rs [![cargo-i][]][cargo-a] [![travis-i][]][travis-a]
========

A streaming compression/decompression library for rust with bindings to liblzma.
[Documentation][docs]

```toml
[dependencies]
xz = "*"
```

### Prerequisites

* [rust][]
* [liblzma][] *or*<br>
  GNU Make and C compiler
