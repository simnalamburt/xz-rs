xz-rs [![version-i][]][crates] [![buildstat-i][]][travis]
========

A streaming compression/decompression library for rust with bindings to liblzma.
[Documentation][docs]

```toml
[dependencies]
xz = "*"
```

* [Rust][]
* [xz][]<br>
