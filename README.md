xz-rs [![crate version]][crate link]
========
A streaming compression/decompression library for [Rust] with bindings to
[liblzma]. It's currently an alias of [`xz2`] crate. See [documentation] for the
further details.

```toml
[dependencies]
xz = "0.1"
```
