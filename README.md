xz-rs [![crates-i][]][crates-a] [![travis-i][]][travis-a]
========

A streaming compression/decompression library for [Rust][] with bindings to
liblzma. [Documentation][doc]

```toml
[dependencies]
xz = "*"
```

### Prerequisites

* [Rust][]
* [liblzma][] *or*<br>
  tar, make, c compiler
